# てにをは英語 学習ポータル構成案

## 目的
- YouTube、NOTE、英単語アプリ、動く英語イメージ図鑑を1つの学習導線として見せる。
- 売り込み感を抑えつつ、学習者が次に何をすればいいか迷わない状態を作る。

## コンセプト
- 「英語学習の入口ページ」
- 商品一覧ではなく、学習者が自分に合う学び方を選べるポータル

## ページ全体像
1. ファーストビュー
2. どんな人向けか
3. 学習ルート案内
4. 各コンテンツの役割紹介
5. よくある悩み別の入口
6. 最後の案内

## 1. ファーストビュー
### 見出し案
英語学習を、動画だけで終わらせない。

### サブコピー案
「てにをは英語」の動画、補足解説、英単語アプリ、動く英語イメージ図鑑をつないで、日本人のための学習ルートを作りました。

## 2. どんな人向けか
### セクション見出し
こんな方に向いています

### 本文案
- 英語をやり直したい
- 単語を覚えてもすぐ抜ける
- 日本語では分かるのに英語で使えない
- 丸暗記ではなく、イメージで理解したい

## 3. 学習ルート案内
### セクション見出し
自分に合うところから始めてください

### ルートA
#### タイトル
まずは気軽に学びたい方へ

#### 導線
YouTube -> NOTE補足解説

#### 説明
まずは動画で全体像をつかみ、必要なところだけNOTEで深掘りします。

### ルートB
#### タイトル
単語力を増やしたい方へ

#### 導線
YouTube -> 英単語アプリ

#### 説明
動画で単語のイメージをつかみ、アプリで繰り返し定着させます。

### ルートC
#### タイトル
イメージで理解したい方へ

#### 導線
YouTube -> 動く英語イメージ図鑑

#### 説明
言葉のニュアンスや動きを、視覚的なイメージで確認できます。

### ルートD
#### タイトル
しっかり継続したい方へ

#### 導線
YouTube -> NOTE -> アプリ -> 図鑑

#### 説明
動画で理解し、NOTEで整理し、アプリで反復し、図鑑で感覚を定着させます。

## 4. 各コンテンツの役割紹介
### YouTube
- 入口
- 英語の考え方をつかむ場所

### NOTE
- 補足解説
- 動画だけでは伝えきれない内容を整理する場所

### 英単語アプリ
- 反復
- 単語を覚えて使える状態に近づける場所

### 動く英語イメージ図鑑
- 感覚の補強
- イメージが重要な単語や熟語を視覚的に理解する場所

## 5. よくある悩み別の入口
### 見出し
悩みから選ぶ

### 入口案
- 単語が覚えられない
- 前置詞やニュアンスが分かりにくい
- 動画を見ても定着しない
- 英語をやり直したいが、何から始めればいいか分からない

## 6. 最後の案内
### 見出し案
今の自分に合うところから始めてください

### 本文案
全部を一度にやる必要はありません。まずは今の課題に合うものを1つ選んで、そこから少しずつ進めてみてください。

## デザイン方針
- LP感よりも、学習ガイド感を優先する
- 強いセールス文言は避ける
- 「購入」より「学び方」「使い分け」を前面に出す
- 1ページ内で次の行動が自然に決まるようにする

## 実装メモ
- 最初は1ページのシンプルな静的ページで十分
- 先に文章と導線を固める
- 反応を見てからデザインや分岐を増やす
